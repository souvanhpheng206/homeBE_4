# 📘 Student Guide: Getting the Project from GitHub (Visual Studio)

## 🎯 Objective

Students will learn how to:

* Download (clone) a project from GitHub
* Open it in Visual Studio
* Run and explore the code

***

## ✅ Prerequisites

Before starting, make sure you have:

* Visual Studio installed ✅
* Git installed ✅
* Internet connection ✅

***

## 📥 Step 1 — Copy Repository URL

1. Go to the project page on GitHub  
   👉 `https://github.com/sarayut-cdti/FileProcessing`

2. Click the green **Code** button

3. Copy the URL:

```
https://github.com/sarayut-cdti/FileProcessing.git
```

***

## 💻 Step 2 — Clone in Visual Studio

1. Open **Visual Studio**

2. On the start screen, click:
   👉 **Clone a repository**

   *(or go to: File → Clone Repository)*

3. In the **Repository location** box, paste:

```
https://github.com/sarayut-cdti/FileProcessing.git
```

4. Choose a local folder (e.g., `Documents`)

5. Click:
   👉 ✅ **Clone**

***

## 📂 Step 3 — Open the Project

After cloning:

1. Visual Studio will open automatically  
   OR manually open the `.sln` file:

```
FileProcessing.sln
```

2. Wait for project to load

***

## ▶️ Step 4 — Run the Project

1. Press:
   👉 **F5** (Run)

   OR click:
   👉 **Start Debugging**

2. The application should start

***

## 🧪 Step 5 — Explore and Practice

Students should:

* Read the source code (`Program.cs`, `TextCSV_Viewer.cs`)
* Try modifying small parts of the code
* Test with different CSV files

***

## ❗ Troubleshooting

### Problem: Cannot clone

* Check internet connection
* Verify Git is installed:
  ```
  git --version
  ```

***

### Problem: Project doesn’t run

* Check build errors
* Ensure correct .NET version installed

***

### Problem: Files missing

* Make sure cloning completed successfully
* Try re-cloning

***

## 💡 Learning Tips

* Start small: modify one function at a time
* Read error messages carefully
* Ask questions when stuck
* Focus on understanding, not just running code
